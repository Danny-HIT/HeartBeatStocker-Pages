You can also open a GitHub issue. A GitHub account is required. Do not include personal or health information in a public issue.

Include These Details

  • Apple Watch model.
  • watchOS version.
  • The workout type you selected.
  • Whether Health access was allowed for Heart Rate and Workouts.
  • Whether Save to Health, vibration cues, repeated feedback, and session time were enabled.
